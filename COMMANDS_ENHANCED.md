## Task Scheduling

Schedule tasks to run automatically at specific times!

### Scheduling Commands

| Command | Description | Example |

| at [time] [task] | Schedule task for specific time | "at 10 AM open project folder" |

```

**Example 2: Schedule music**
```

**Example 3: List tasks**
```

- **12-hour format:** "at 10 AM", "at 2:30 PM"

## Tips for Best Results

### Voice Recognition Tips

1. **Speak Clearly:** Use normal speaking volume and pace
2. **Use "Gideon":** Include "Gideon" in commands for better recognition
3. **Reduce Noise:** Minimize background noise
4. **Wait for Response:** Let Gideon respond before giving next command
5. **Check Microphone:** Ensure microphone is working properly

### Command Tips

1. **Natural Language:** Gideon understands variations
   - ‚úì "open chrome" = "chrome kholo" = "start chrome"
   - ‚úì "what time is it" = "time batao" = "tell me the time"

2. **Workflows:** Use workflows for repetitive multi-step tasks
   - Instead of: "open chrome", "open vs code", "open notepad"
   - Say: "start my workday"

3. **Scheduling:** Schedule recurring tasks
   - "at 9 AM start my workday"
   - "at 2 PM take a break"

4. **Confirmation:** Critical actions require confirmation
   - Gideon will ask "Are you sure?" for destructive operations
   - Say "yes" to confirm, "no" to cancel

### Exit Commands

**Most Reliable Exit Commands:**
- "quit gideon"
- "exit gideon"
- "gideon quit"

**Note:** Simple "quit" or "exit" also work, but using "gideon" is more reliable.

## Advanced Features

### Confirmation Prompts

For safety, Gideon asks for confirmation before:
- Deleting files
- Cleaning downloads folder
- Clearing scheduled tasks
- Other destructive operations

**Example:**
```
You: "clean downloads"
Gideon: "Are you sure you want to clean your downloads folder? Say yes or no."
You: "yes"
Gideon: "Confirmed. Proceeding."
        "Organizing 25 files"
        "Done! Organized 25 files."

## Troubleshooting

### Command Not Recognized

**Problem:** "I didn't catch that" or unknown command

**Solutions:**
1. Speak more clearly
2. Reduce background noise
3. Check microphone settings
4. Try command variation
5. Say "help" to see available commands

### Shutdown Not Working

**Problem:** "quit gideon" not stopping Gideon

**Solutions:**
1. Say "quit gideon" (with "gideon")
2. Try "exit gideon"
3. Try "goodbye gideon"
4. Simple "quit" or "exit" as fallback

### YouTube Not Playing

**Problem:** Videos not opening

**Solutions:**
1. Check internet connection
2. Be specific: "play [exact name] on youtube"
3. Try: "youtube [song name]"
